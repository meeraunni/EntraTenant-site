Sent mail from a shared mailbox shows in personal Sent Items

Enable mailbox copy settings so a shared mailbox keeps sent copies in its Sent Items folder.

2026-03-078 minExchange Online
Back to Blog

This behavior is common with delegated sends. If you need a sent copy in the shared mailbox for continuity and audit, enable mailbox copy settings.

Fix

# Connect-ExchangeOnline

$SharedMailbox = "shared@domain.com"

Set-Mailbox -Identity $SharedMailbox `
  -MessageCopyForSentAsEnabled $true `
  -MessageCopyForSendOnBehalfEnabled $true

Verify

Send a test message from the shared mailbox and confirm the message appears in the shared mailbox Sent Items folder.
