MICROSOFT DEFENDER FOR OFFICE 365

Remove an email across mailboxes using Microsoft Defender

Use Explorer to scope the message, run remediation, review approvals, and verify removal.

2026-03-079 minDefenderEmail
Back to Blog

If a message was delivered and needs to be removed across mailboxes, use Defender remediation. Outlook recall is not reliable across clients and recipients.

Before you start Collect at least one stable identifier. Internet Message ID is best. If not available, use sender, subject, and a narrow time range.

Scope the message in Explorer

Use Explorer to locate the message and confirm delivery details: recipients, folder, and any user interaction signals.

Run a remediation action

Select the message and run the appropriate action. For confirmed malicious content, remove it. For investigation, choose a less destructive action and continue review.

Review approvals (AIR)

If Automated Investigation and Response is enabled, actions may be pending approval. Remediation is not complete until the status shows completed successfully.

Verify

Verify in Defender action status and in Explorer results. If the message remains, confirm you targeted the correct artifact and recipients before repeating actions.
